# Ochi – React Landing Page Clone

A modern, animated landing page inspired by the Ochi design studio website, built using React, Vite, and Tailwind CSS.
This project focuses on creating smooth UI interactions, responsive layouts, and visually appealing sections.


## 🚀 Features

Responsive modern UI
Smooth animations and transitions
Component-based React architecture
Tailwind CSS styling
Clean and modular folder structure
Interactive sections and hover effects

## 🛠️ Tech Stack

### Frontend
React
Vite
Tailwind CSS
JavaScript (ES6+)

### Tools
Git
GitHub
VS Code

## ⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/Achintxv/React.git
2. Navigate to project folder
cd React/Ochi
3. Install dependencies
npm install
4. Run the development server
npm run dev

The app will run on:

http://localhost:5173
