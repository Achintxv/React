    <h1>Ochi – React Landing Page Clone</h1>

    <p>A modern, animated landing page inspired by the Ochi design studio website, built using React, Vite, and Tailwind CSS. This project focuses on creating smooth UI interactions, responsive layouts, and visually appealing sections.</p>

    <h2>🚀 Features</h2>
    <ul>
        <li>Responsive modern UI</li>
        <li>Smooth animations and transitions</li>
        <li>Component-based React architecture</li>
        <li>Tailwind CSS styling</li>
        <li>Clean and modular folder structure</li>
        <li>Interactive sections and hover effects</li>
    </ul>

    <h2>🛠️ Tech Stack</h2>
    <h3>Frontend</h3>
    <ul>
        <li>React</li>
        <li>Vite</li>
        <li>Tailwind CSS</li>
        <li>JavaScript (ES6+)</li>
    </ul>

    <h3>Tools</h3>
    <ul>
        <li>Git</li>
        <li>GitHub</li>
        <li>VS Code</li>
    </ul>

    <h2>⚙️ Installation & Setup</h2>
    <ol>
        <li><strong>Clone the repository</strong><br>
            <code>git clone https://github.com/Achintxv/React.git</code>
        </li>
        <li><strong>Navigate to project folder</strong><br>
            <code>cd React/Ochi</code>
        </li>
        <li><strong>Install dependencies</strong><br>
            <code>npm install</code>
        </li>
        <li><strong>Run the development server</strong><br>
            <code>npm run dev</code>
        </li>
    </ol>
    <p>The app will run on: <code>http://localhost:5173</code></p>
